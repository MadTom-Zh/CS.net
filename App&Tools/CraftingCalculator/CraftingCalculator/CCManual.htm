快速上手

双击运行主程序CraftingCalculator.exe：

1.       点击右侧地球图标，选择适合的界面语言；

2.       在窗口左侧选择场景；

3.       点击手动计算进入自定义计算模式；

4.       或者点击自动计算，针对某个产品自动计算生产线；

手动计算模式

图形区域基本操作

         鼠标中间按住图形空白区域后，拖动鼠标，移动整个图形；

         滚动鼠标滚轮，缩放整个图形；

         鼠标左键按住节点标题后，拖动鼠标，移动节点位置；

         鼠标左键按住节点右下角，拖动鼠标，调整节点大小；

         点击节点，或点击线条，选中单个节点或线条；

         按住Shift后点击，追加选择节点或连线；

         按住Alt后点击，排除选择节点或连线；

         在空白处按住鼠标左键后，拖动鼠标，框选范围内的所有节点和连线；

         按住Shift后，拖框，追加选择；

         按住Alt后，拖框，排除选择；

选取需要的制程

1.       双击查找物品图标，选择一个物品，选择物品后，下方列出制造这种物品的所有制程（如果没有，则列出可能的原料制程）；

2.       点选查看制程需要的原料和产品，制程本身显示第一产品的产出速度，单位分别为ips，ipm，iph；

3.       通过拖拽制程，将节点放置在右侧画图区域中，通过拖拽标题区域，可以移动节点，拖拽右下角尺寸拖柄改变尺寸；

4.       通过双击制程的某个原料，列出制造这个原料的所有制程，进一步查看这个原料如何制造；

5.       通过双击制程的某个产品，列出以此为原料的所有制程，查看这个产品可进一步制造什么；

*通过按下键盘的Backspace，可回溯之前查看的制程列表；

如何使用物品选取窗口

1.       键入物品名称后，按下键盘Enter可执行搜索，效果和点击右侧搜索按钮效果相同；

2.       通过点击搜索按钮，查询含有此关键字的物品，效果和连续按下键盘Enter效果相同；

3.       如果某些物品设置有类型，也可通过类型搜索同类型的相关物品（相当于tag标记）；

4.       左侧会列出全部物品，也可以通过浏览来手动选择；

5.       最后选择好物品后，点击好的按钮，或使用快捷键Ctrl+Enter，效果相同（或点击取消，或关闭窗口，同取消）；

*搜索时可使用模糊匹配，如输入ir re，也可以快速找到iron rebar；

*注意搜索后如果再次点击搜索输入框，或修改输入内容，将打断连续搜索状态，进入新搜索状态；

连接并完善生产线

1.       默认为选择模式，在此模式下可以选择节点，移动节点，重设节点尺寸，选择连线，通过右击鼠标来操作节点和连线；

2.       通过设置通道工具栏，可以收起一些用不到的通道，方便用户操作，如只显示传送带mk4，水管mk2和电线；

3.       点击通道按钮后，将进入连线模式，此模式下可以连接节点，但不能移动和改变节点尺寸；

4.       连线后，如果连线为黑色，则表示两端供需匹配，否则显示为红色；

5.       图形配置好后，可在描述栏中写入需要的文字；

6.       点击保存进行快速保存，如果是第一次存档，则会要求输入图形名称，效果同按钮另存为；

7.       点击另存为，将当前图形保存到新存档，需要输入新图形名称；

8.       通过双击图形列表中的条目，加载已经存档的图形，加载的图形，列表中会以橙色框选，并显示当前图形名称；

9.       通过点击删除按钮，删除已加载的图形；

10.   通过点击清理按钮，将当前图形区域清空，并取消已选择图形存档的选择状态；

*连线状态下，如果点击节点无效果，请检查选择的节点是否支持输出物品，如传送带点击流体输出将无效果；

*连线状态下，如果无法连接两个节点，请检查两端物品是否一致；

*按下Ctrl+S执行保存命令，和点击保存按钮效果相同；

*（目前）只有加载的图形存档后，才可删除此存档；

右击菜单

1.       设置倍数，可设定单个节点、单条连线的倍数；

2.       删除节点，删除选中的所有节点；

3.       删除连线，删除选中的，或连至节点的所有连线；

4.       匹配来源，将当前连线的倍数匹配到左侧节点输出速度相同的倍数；

5.       匹配目标，将当前连线的倍数匹配到右侧节点输入速度相同的倍数；

6.       自动完成，尝试匹配所有节点和连线的倍数，并将需要的输入和输出列入输入输出栏目中；

*在选择单个、多个，节点、连线的不同情况，菜单可用项目也会不同；

*删除操作不会二次确认，执行删除前请仔细查看；

设定节点倍数

 

1.       数量区域，显示当前节点需要的机器数量，可设置新数量；

2.       两边显示个端口的速度，接近中间的是端口速度，修改速度后，会联动机器数量和其他端口速度；

3.       通过点击配对按钮，将节点数量设置到匹配这个端口外连线速度的数量；

4.       靠外侧的数字为外连线速度，如果显示为黄色，则意味着对方节点存在多个连线；

5.       窗口顶端可设定速度单位，按每秒数量，每分钟或每小时；

6.       设定好后，点击好的按钮，确认修改；

7.       应用到全部，则在修改当前节点后，尝试以当前节点数量来匹配关联整张图（相连的节点和连线）的数量；

*如果节点的端口和连线速度不匹配，则显示为常规色，输入橙色，输出蓝色，如果匹配，则为绿色；

*如果通过键盘设置数量，输入栏将变为黄色，输入后，请按下Enter确认；

*数量输入栏支持计算公式，如1+2/(3*4 )；

设定连线倍数

 

1.       点击匹配来源，将连线数量匹配到左侧节点输出端口速度一致的数量（与右击菜单的匹配来源功能相似，但需要手动确认）；

2.       点击匹配目标，将连线数量匹配到右侧节点输入端口速度一致的数量（与右击菜单的匹配目标功能相似，但需要手动确认）；

*其他功能于设置节点速度的一致；

全局空速

1.       点击节点端口，将在左侧显示端口的速度；

2.       点击节点本体，将显示机器数量；

3.       点击连线，显示连线数量；

4.       通过修改数量，联动全图节点和连线的数量和速度；

5.       点击重置，复原设定新速度之前的速度；

6.       点击设定到基准，将新设定的全局速度应用到全部节点和连线（应用后，用户可保存存档）；

自动计算模式

*因为每增加一个工序，组合可能性就会翻倍，也就是说如果生产线存在12个工序，组合数量可能会达到1024种，所以如果设定了大量过滤之后还无法成功计算的话，建议还是使用手动模式；
